# 切换到管理员状态

以下的操作都是基于`Windows`，如果你使用其他系统，请自行解决

安装`Win`和`X`键，然后选择管理员终端即可

![image-20211210201040357](https://ooszy.cco.vin/img/blog-note/image-20211210201040357.png?x-oss-process=style/pictureProcess1)

然后选择`Windows终端 (管理员)`即可
